1.To check remote Repo is mapped.



***git remote -v***



2.To add remote repo.



**git remote add origin URL.git**



3.To verify the remote repo is added.



**git remote -v**



4.To initialize the folder for GIT



**git init**



5.To check the status of the current.



**git status**



6.To remove all the changes in the current and provide a new fresh start.



**rm -rf .git**



7.Unstage everything from the folder



**git restore --staged .**



8.Changing the branch name from master to main\*\*.\*\*



**git branch -m master main**





**GIT Merge  --Merges the code from branch to main**

**GIT Stash  --saves the code temporarily without commit**

**GIT Rebase --Detaching the branch and attached to new starting point**
